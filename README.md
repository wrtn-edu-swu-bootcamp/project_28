# Design Bridge 🎨

그래픽 디자인 전공 학생들을 위한 학술-실무 언어 번역 서비스

![Next.js](https://img.shields.io/badge/Next.js-14-black)
![TypeScript](https://img.shields.io/badge/TypeScript-5-blue)
![Tailwind CSS](https://img.shields.io/badge/Tailwind-3-38bdf8)

## ⚠️ 현재 버전 안내

**이 프로젝트는 현재 로컬 데이터 기반으로 작동합니다**
- ✅ **API 키 불필요** - 별도의 설정 없이 바로 사용 가능
- 📚 **학술 용어 50개 + 실무 용어 50개** - 총 100개 핵심 용어 완벽 정리!
- 🔄 **양방향 번역** - 학술 ↔ 실무 완벽 지원
- 📖 **구체적인 설명** - 학생들이 실무에서 바로 활용할 수 있도록 상세 설명
- 🔍 **사전 형태** - 모든 용어를 한눈에 확인
- 🖱️ **클릭 한 번** - 용어 클릭으로 바로 검색

> 💡 향후 AI API 연동 시 무제한 용어 번역 지원 예정

## 📖 서비스 소개

Design Bridge는 그래픽 디자인 전공 학생들이 학교에서 배운 학술 용어를 실무 언어로 이해하고, 반대로 실무에서 쓰이는 용어의 이론적 배경을 파악할 수 있도록 돕는 번역 서비스입니다.

### 주요 기능

- ✅ **양방향 번역**: 학술 ↔ 실무 용어 번역
- 💡 **실무 맥락 제공**: 실제 업무 상황에서의 사용 예시
- 📚 **용어 사전**: 30개 핵심 디자인 용어 탑재
- 🔍 **검색 히스토리**: 최근 검색한 용어 저장
- 📱 **반응형 디자인**: 모바일, 태블릿, 데스크톱 지원
- ⚡ **즉시 사용**: API 키 설정 불필요

## 🚀 시작하기

### 필수 요구사항

- Node.js 18 이상
- npm 또는 yarn

### 설치 및 실행

1. **저장소 이동 및 의존성 설치**

```bash
cd Final_pj
npm install
```

2. **개발 서버 실행**

```bash
npm run dev
```

3. **브라우저에서 열기**

http://localhost:3000 접속

**끝! 🎉 API 키 설정 없이 바로 사용하세요!**

## 📚 포함된 용어

### 학술 용어 (50개) 📖

**디자인 원리 (13개)**: 시각적 위계, 게슈탈트 법칙, 근접성의 원리, 유사성의 원리, 연속성의 원리, 폐쇄성의 원리, 대칭, 비대칭, 균형, 반복, 강조, 대비, 통일성

**타이포그래피 (7개)**: 타이포그래피, 가독성, 자간, 행간, 위계, 서체 선택, 커닝

**색채 이론 (10개)**: 색상환, 색채 대비, 채도, 명도, 색온도, 보색, 유사색, 색의 심리, 그레이스케일, 컬러 팔레트

**레이아웃 (10개)**: 여백, 그리드 시스템, 비율, 마진과 패딩, 정렬, Z-인덱스, 반응형 레이아웃, F-패턴과 Z-패턴, 가시 영역, 컨테이너

**디자인 요소 (5개)**: 질감, 형태, 선, 아이콘, 이미지

**UX/UI (5개)**: 인터페이스 디자인, 사용자 경험, 인터랙션, 접근성, 반응형 디자인

### 실무 용어 (50개) 💼

**디자인 프로세스 (8개)**: 레퍼런스, 시안, 러프, 픽스, 컨펌, 리비전, 피드백, 목업

**타이포그래피 (5개)**: 폰트, 자간, 행간, 본문, 헤드라인

**색상 (6개)**: 컬러, 톤앤매너, 무드, 비비드, 파스텔톤, 다크모드

**레이아웃 (10개)**: 여백, 간격, 마진, 패딩, 그리드, 가이드라인, 정렬, 센터, 밸런스, 비율

**디자인 원리 (8개)**: 포인트, 강조, 시선 흐름, 우선순위, 그룹감, 통일감, 대비, 리듬감

**브랜딩 (4개)**: 브랜딩, BI, 로고, CI

**UI/UX (6개)**: UI, UX, 반응형, 모바일 대응, 버튼, 아이콘

**기타 (3개)**: 작업물, 누끼따기, 텍스처

### ✨ 총 100개 용어 완벽 정리! 

각 용어마다:
- 정확한 학술적 정의
- 실무 현장에서 쓰이는 표현
- 학생들이 이해하기 쉬운 구체적 설명
- 3가지 실무 상황별 예시 (디자인 회의, 클라이언트 피드백, 업무 지시)

## 💡 사용 방법

### 1. 용어 사전에서 선택
- 화면의 "사용 가능한 용어" 섹션에서 원하는 용어 클릭
- 자동으로 검색창에 입력됨

### 2. 직접 검색
- 검색창에 알고 싶은 용어 입력
- "번역하기" 버튼 클릭 또는 Enter

### 3. 방향 전환
- 상단의 "학술 → 실무" 버튼으로 번역 방향 전환
- 학술 용어를 실무 용어로, 또는 그 반대로

### 4. 결과 확인
- **학술 용어**: 정식 명칭
- **실무 용어**: 현장에서 사용되는 표현들
- **설명**: 두 용어의 관계 이해
- **실무 예시**: 실제 상황에서의 사용 예시

## 🎯 주요 특징

### 1. 즉시 사용 가능
- API 키 설정 불필요
- 복잡한 환경 설정 없음

각 용어마다 3가지 실무 상황 제공:

- 깔끔하고 직관적인 인터페이스

- 클릭 한 번으로 검색
- 반응형 디자인으로 모든 기기 지원

### 4. 학습 도구로 최적화
- 이론과 실무의 연결고리 명확히 제시

- 검색 히스토리로 복습 가능
